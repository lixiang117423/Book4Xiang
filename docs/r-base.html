第 1 章 语言基础

1.1 R的安装

  • 各系统下R的安装，点击访问。
  • 安装最新版R
sudo sed -i "s/# deb-src/deb-src/g" /etc/apt/sources.list
sudo apt-get update
sudo apt-get build-dep r-base-dev

cd ~/Downloads
wget -c https://cran.r-project.org/src/base/R-4/R-4.1.0.tar.gz
tar -xf R-4.1.0.tar.gz
cd R-4.1.0
./configure
make -j9
sudo make install
  • 安装R包报错：ERROR: failed to lock directory ‘/usr/local/lib/R/library’ for modifying的解决方案：
R CMD INSTALL --no-lock <pkg>
install.packages("Rcpp", dependencies = TRUE, INSTALL_opts = '--no-lock')

1.2 包的安装更新

  • 更新所有R包
#安装包
install.packages("rvcheck")
#加载包
library(rvcheck)
#检查R是否有更新
rvcheck::check_r()
# 更新所有R包
rvcheck::update_all(check_R = FALSE,which = c("CRAN","BioC","github")
